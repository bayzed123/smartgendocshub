Overview

SmartGen is built on a privacy-first architecture where most tools operate locally on the user's device. We prioritize transparency, minimal data usage, and full compliance with Google Play policies, GDPR, and CCPA.

Data Collection & Usage

  • No personal data (name, email, phone) is collected
  • No user content is stored on servers
  • Only limited technical data is processed
  • Analytics via Firebase & Google Analytics
  • Ads via Google AdMob

External API Processing

Some tools may use external APIs. These are only triggered by user actions. Data is processed temporarily and not stored by SmartGen.

Children’s Privacy

SmartGen is not intended for users under 13. We do not knowingly collect data from children. AdMob is configured with child-safe restrictions.

Analytics & Tracking

  • Firebase Crashlytics for stability
  • Google Analytics for usage insights
  • Google Tag Manager for event tracking
  • All data is aggregated & anonymized

Advertising Disclosure

We use Google AdMob to keep tools free. Users can opt out of personalized ads via device settings. We do not control AdMob targeting systems.

Account Policy

No registration or login is required. SmartGen does not create user accounts or profiles.

Security & Architecture

  • HTTPS encrypted communication
  • Local-first processing model
  • No centralized database
  • Secure API handling

GDPR & CCPA Compliance

  • Right to access (limited to technical data)
  • Right to deletion (via uninstall/cache clear)
  • No sale of user data
  • No behavioral profiling
