What is a Robots.txt File?

A robots.txt file tells search engine crawlers which pages or files the crawler can or can't request from your site. This is used mainly to avoid overloading your site with requests; it is not a mechanism for keeping a web page out of Google.

How to Use

Define your sitemap URL to help bots find your content faster. Add any directories or pages you want to keep private from search results (like admin panels or temporary files) in the disallowed paths section.

FAQs

Where should I upload this file?

Upload the robots.txt file to the root directory of your website (e.g., example.com/robots.txt).

Does robots.txt hide pages from users?

No, it only provides instructions for automated crawlers. Users can still access the pages if they have the URL.

Can I block all bots?

Yes, by using User-agent: * and Disallow: /, but this will stop your site from appearing in search results.
