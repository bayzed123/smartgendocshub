What is a Mailto Link?

A 'mailto' link is a type of HTML link that activates the default mail client on the user's computer or mobile device to send an email. It can include a recipient, subject, and even a pre-written message body.

How to Use

Enter the destination email address and optionally fill in the subject and body fields. The tool will generate the 'mailto:' URI and the corresponding HTML code for you to paste into your website.

Features

  • Automatic URL encoding
  • Support for subject and body
  • Instant HTML code generation
