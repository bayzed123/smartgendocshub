# ⚡ SmartGen: All-in-One Digital & Web Utility Platform

[![Auto Changelog Status](https://github.com/bayzed123/SmartGenQR.oi/actions/workflows/auto-changelog.yml/badge.svg)](https://github.com/bayzed123/SmartGenQR.oi/actions/workflows/auto-changelog.yml) [![Pages Build Deployment](https://github.com/bayzed123/SmartGenQR.oi/actions/workflows/pages/pages-build-deployment/badge.svg)](https://github.com/bayzed123/SmartGenQR.oi/actions/workflows/pages/pages-build-deployment)

**SmartGen** is a comprehensive, open-source web utility platform designed for developers, marketers, and everyday users. It features over 45+ fast, secure, and 100% free tools, all processed entirely on the client-side to ensure maximum privacy and speed.

## ⚙️ Setup & Development

To run SmartGen locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/bayzed123/SmartGenQR.oi.git
    cd SmartGenQR.oi
    ```
2.  **Install dependencies:**
    ```bash
    pnpm install
    ```
3.  **Run build scripts (for blog):**
    ```bash
    node scripts/build-blog.js
    ```
4.  **View locally:** Open `index.html` in your browser.

For a full guide, see the [Setup Guide](WIKI_Setup_Guide.md).
